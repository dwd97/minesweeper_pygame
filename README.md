# Zápočtový program

## Specifikace

Program je kopie hry [Minesweeper](<https://en.wikipedia.org/wiki/Minesweeper_(video_game)>). Je to hra, kde se hráč snaží odkrýt všechna políčka, kde se nenachází mina v co nejkratším čase.

## Instalace a spuštění

Program se spustí příkazem: `python main.py`

Detailnější postup instalace a spuštění [zde](docs/user.md)
